# UTAULARIA (Em progresso)

Galeria colaborativa de voicebanks UTAU criados por artistas brasileiros. Um espaço para descobrir, baixar e celebrar a produção nacional e/ou lusófona de voicebanks.
> Por favor, leve em consideração que quem iniciou este repositório (xiao) está fazendo tudo sozinho, tendo isso em mente saiba que nas versões iniciais não haverão TODOS os voicebanks e o software pode conter falhas.
---

## Sobre o Projeto

UTAULARIA é uma projeto de xiao, fundador do [Studio POMAR](https://github.com/studiopomar) com o objetivo de centralizar e facilitar o acesso aos voicebanks desenvolvidos pela comunidade brasileira de UTAU. Aqui você encontra vozes feitas com carinho, cada uma com sua história e características únicas.

---

## O que você encontra aqui

- Voicebanks nacionais organizados e de alto padrão.
- Informações técnicas detalhadas sobre cada banco de voz.
- Links oficiais para download.
- Créditos aos criadores e documentação.
- Curadoria feita com atenção à qualidade e originalidade.

---

## Como Contribuir

A UTAULARIA é um projeto aberto e colaborativo. Se você criou um voicebank ou conhece algum que merece estar aqui, sua contribuição é bem-vinda.

Para adicionar um novo voicebank:

1. Verifique se ele atende aos critérios de inclusão.
2. Abra uma issue ou pull request com as informações.
3. Aguarde a revisão e aprovação.

Toda contribuição será devidamente creditada.
